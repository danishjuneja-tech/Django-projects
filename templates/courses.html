Beginner Level

A Basic Level course for kids to create stories, animations, and 2D games, learn the basics of Robotics, 3D Design & Sequencing.

Intermediate Level

An Intermediate Level course for kids to explore the working of Sensors, enjoy building Robots, and create exciting Machine Learning projects.

Advanced Level

An Advanced Level course where kids create OpenCV-integrated Robots, 3D Game Development, Web Designing, and Drone Programming.
